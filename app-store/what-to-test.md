# What to Test - Build 18

## 本轮主要改动
- 修复时间选择器过早关闭：选完小时后 picker 不再立即关闭，可以继续选分钟，点"完成"才提交
- 修复推送通知不生效：sound 字段从 boolean 改为 'default'，编辑模式也确保权限已获取

## 重点测试
1. **时间选择器**：添加药物 → 点时间 → 滚动选小时 → 确认 picker 不关闭 → 继续选分钟 → 点"完成"
2. **推送通知**：添加药物设置提醒时间为当前时间 +2 分钟 → 等待通知是否弹出
3. **编辑药物的时间选择**：编辑已有药物 → 修改提醒时间 → 确认时间更新正确

## 已知风险
- 推送通知在模拟器上行为可能与真机不同，建议在真机上验证
