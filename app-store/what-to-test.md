2. **高对比度**（Pro）：设置 → 显示 → 开启高对比度 → 回 Today/药物页确认次要文字颜色加深、边框加粗
3. **时间选择器**：添加药物 → 点时间 → 选完小时后 picker 不关闭 → 选分钟 → 点完成
4. **推送通知**：添加药物设提醒 → 确认到时收到通知
