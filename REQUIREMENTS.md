# ElderEase - 老人药物提醒 App

## 产品定位
给爸妈用的吃药提醒 -- 大字体、零广告、不要订阅、数据不出手机。

## 目标用户
- 主要：60+ 老年人（直接使用者）
- 次要：30-50 子女/护理者（购买者+安装者）

## MVP 功能清单

### 核心功能
1. **药物管理** - 添加/编辑/删除药物（名称、剂量、频率、提醒时间）
2. **本地推送提醒** - 到点推送通知，大字体显示药名+剂量
3. **一键标记服药** - 整个卡片可点，大按钮"已服药"
4. **服药历史** - 简单日历视图，绿色=已服，红色=漏服，灰色=未到

### 辅助功能
5. **今日药物总览** - 首页只显示今天该吃的药和状态
6. **库存提醒**（可选 V1.1）- 剩余药量追踪，快没了提醒补药

### 无障碍
7. **大字体** - 默认 20sp+，支持 Dynamic Type
8. **高对比度** - WCAG 2.1 AA 合规
9. **VoiceOver 完整支持**
10. **触控目标 48x48dp 最小**

## 屏幕列表

| # | 屏幕 | 说明 |
|---|------|------|
| 1 | 今日药物（首页） | Tab 1，当天药物列表+服药状态 |
| 2 | 所有药物 | Tab 2，全部药物管理 |
| 3 | 服药历史 | Tab 3，日历视图 |
| 4 | 设置 | Tab 4，提醒偏好、字体大小、关于 |
| 5 | 添加/编辑药物 | Modal，表单页 |
| 6 | Paywall | 免费3药+买断$9.99无限 |
| 7 | Onboarding | 首次启动引导（1-2页，极简） |

## 设计方向
- **温暖、圆角、高对比度**，不要冷色调医疗感
- 品牌色：温暖橙/琥珀色系（传递安心、温暖感）
- 大圆角卡片，充裕间距
- 全局禁止 emoji，图标用 lucide 矢量线条
- 图标需柔和背景色圆角方块衬底

## 技术约束
- Expo SDK 55 + expo-router
- 纯本地存储（expo-secure-store 或 AsyncStorage）
- expo-notifications 本地推送
- 12 种语言 i18n
- RevenueCat 买断制 IAP
- 不需要后端

## 变现方式
- 免费版：最多追踪 3 种药物
- 买断 $9.99：无限药物
- 无广告、无订阅
